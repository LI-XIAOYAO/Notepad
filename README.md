# Notepad

> 基于 [AntdUI](https://github.com/AntdUI/AntdUI) 实现的一款简单的记事桌面程序

- 云存储，目前依赖邮箱，邮箱文件夹 `Notepad`，测试 `163` `QQ` 可用其他自测
- 支持数据加密
- PIN
- 自定义主题
